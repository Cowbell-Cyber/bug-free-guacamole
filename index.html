Instructions

You're going to make a simple notifications application. In short it will make an api request that fetches the latest notifications, as well as allowing the user to dismiss individual notifications.

Requirements

  1. Use the fake function app.getNotifications to get the list of notifications. important
  2. Talk through your thoughts while working. important
  • Notifications should be located in the upper right corner of the screen.
  • If more than one notification is present it should show up stacked one on top of the other.
  • The notifications should be sorted by their timestamp. (e.g. newest is at the top, and oldest is at the bottom)
  • Each notification should have a human-readable timestamp printed somewhere in the notification.
  • Each notification should have a dismiss button that allows the user to hide a specific notification.
  • All notifications should have some similar styling to make it easy to see.

Bonus

  • Try to match the style of the page or at least make the notifications look "good".
